List of HTML tags

  1. html

    The root element of an HTML document. Contains all other HTML elements on the page.

  2. head

    Contains meta-information about the HTML document, such as the title and links to external resources. Elements inside head are not displayed on the webpage.

  3. title

    Specifies the title of the HTML document, which appears in the browser's title bar or tab.

  4. meta

    Provides metadata about the HTML document, such as character encoding and viewport settings.

  5. link

    Used to link external resources like stylesheets to the HTML document.

  6. script

    Embeds or references JavaScript code for enhancing interactivity on the webpage.

  7. style

    Contains CSS rules for styling HTML elements within the document.

  8. body

    Contains the visible content of the webpage, including text, images, and other elements.

  9. h1 to h6

    Headings of varying levels, with h1 being the highest and h6 the lowest. Used to structure and emphasize content hierarchically.

  10. p

    Defines a paragraph of text. Commonly used for structuring and formatting text content.

  11. a

    Creates hyperlinks to other web pages or resources, enhancing navigation.

  12. img

    Embeds images within the document, enhancing visual content.

  13. table

    Defines a table for organizing and displaying data.

  14. form

    Creates a container for user input elements and defines how data is sent to a server.

  15. input

    Represents various types of input fields within a form, such as text, checkboxes, or radio buttons.
